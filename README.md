# Image View
Simple application to view images for OS Windows.

## Features
* Can load image at file path "D:/Vlad/Pictures/15047665988060.jpg" or crash otherwise.

## Image formats
* BMP
* GIF
* ICO
* JPEG
* PNG
* TIFF
* WMP
* DDS

## Requirements
* Windows 7 or Windows 8 or Windows 10
* DirectX 11 support
* Direct2D support
* Visual Studio 2015 C++ Redist
